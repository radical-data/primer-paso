# `@primer-paso/certificate`

Shared certificate model and PDF-generation package for Primer Paso.

This package owns:

- vulnerability certificate draft types
- organisation issue request types
- runtime validation helpers
- fixture data for tests
- official PDF template location
- PDF generation contract

It does not own:

- public user journey screens
- organisation authentication
- organisation authorisation
- audit logs
- email delivery
- persistence

Those belong to the public app, organisation portal, and database/auth packages.

## Official template

The official PDF template must be placed at:

```txt
packages/certificate/templates/certificado-vulnerabilidad.pdf
```

Do not replace it with a reconstructed PDF unless an ADR explicitly accepts that.

## PDF generation

The current generator is a proof implementation for a flat PDF template. It loads the
official template, draws the certificate data onto the first page, sets inspectable PDF
metadata, and returns the generated bytes.

This is intentionally not yet a fielded-template implementation. A future PR may replace
coordinate drawing with AcroForm fields by committing an official-template derivative with
stable field names.

To export a sample PDF from the committed fixture:

```sh
pnpm --filter @primer-paso/certificate export:sample-pdf
```

## Tests

The package includes schema tests and PDF-generation contract tests.
